# 🧩 ParallelAnalyzer

**ParallelAnalyzer** — это аналитическая лаборатория для студентов, позволяющая изучать и сравнивать различные подходы к параллельным вычислениям на платформе .NET.  
Проект позволяет выполнять BenchmarkDotNet-тесты для разных типов задач (числовых и файловых), сохранять результаты в базу данных SQLite и визуализировать их в виде графиков.

---

## 📚 Основные возможности

- Анализ производительности 13 параллельных паттернов:
  - `Sequential`, `Parallel.For`, `PLINQ`, `TasksByCores`, `Parallel.Invoke`,  
    `Parallel.ForEach`, `Parallel.ForEachConcurrentBag`, `ParallelForList`,  
    `ListProcessing`, `TasksLists`, `ArrayPoolSimulated`, `PLinqWithDegree`, `Partitioner.ForEach`.
- Визуализация результатов с помощью **ScottPlot.Eto**
- Сохранение данных в **SQLite** через **Entity Framework Core**
-  Просмотр и фильтрация сохранённых сессий:
  - фильтры по процессору, задаче и количеству ядер;
  - сортировка по дате и ID;
-  Выбор типа задачи:
  - **Анализ числовых методов** (поиск простых чисел, сортировка, разворот массива, чередование чёт/нечёт и др.);
  - **Анализ файловых методов** (поиск часто встречающегося пользователя, удаление дубликатов);
-  Сохранение графиков результатов в формате `.jpg`.

## Используемые технологии

| Компонент | Назначение |
|------------|------------|
| **C# / .NET 8.0** | Основной язык и платформа разработки |
| **Eto.Forms** | Кроссплатформенный графический интерфейс |
| **ScottPlot.Eto** | Отрисовка графиков производительности |
| **BenchmarkDotNet** | Проведение тестов производительности |
| **Entity Framework Core (SQLite)** | Работа с базой данных |
| **DB Browser for SQLite** | Визуальный просмотр базы данных (опционально) |

## Примечания для запуска проекта:
 - При разработке использовалас **Visual Studio 2022**  с поддержкой **.NET 8.0**.
 - Откройте решение ParallelAnalyzer.sln.
 - Установите проект ParallelAnalyzer.UI как запускаемый.
 - Нажмите запустить.

## Таблицы базы данных (SQLite)

Systems — сведения о системе (CPU, ОЗУ, ОС)

Tasks — описание тестовых задач

Sessions — сессии тестов (дата, система, задача)

NumberBenchmarks — результаты числовых тестов
